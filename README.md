## Sugerencias :heavy_exclamation_mark:
1. Guarde datos **Basura**
  a. Apropiado aplicar este comando > dotnet new gitignore
2. Errores de salida de datos es decir :no_entry_sign:
  b. Las sugerencias del **ERROR CS8600 / CS8604**
      1. **CS8600**
           a. > Los datos que se volvian en la caja se retomaba ya con un valor y la sugerencia era de un cambio si el usuario no cumplia con las indicaciones convirtiendo en **ERROR**
      2. **CS8604**
           b. > Se somete cuando el **USUARIO** ingresa números en vez de letras
 - - - 
  ##### Solucuines 
  Lo recomendable es buscar las execepciones adecuada para una mejor salida del dato como el uso del Try caht y entre otros extensiones que ayuda alojar los errores.
